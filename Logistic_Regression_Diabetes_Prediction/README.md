# Logistic Regression - Diabetes Prediction

## Description
This project predicts whether a person has diabetes using Logistic Regression.

## Algorithm
- Logistic Regression

## Dataset
- diabetes.csv

## Output
- 0 : Non-Diabetic
- 1 : Diabetic

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
