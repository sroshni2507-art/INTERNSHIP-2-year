# Brain Tumor Detection using CNN

## Dataset
MRI images from Kaggle (not included in repo).

## Model
CNN model file (.h5) is not included due to GitHub size limits.
You can re-train the model using the provided notebook.

## How to Run
1. Download the dataset from Kaggle
2. Run the notebook to train the model
3. Place 'brain_tumor_model.h5' in streamlit_app/
4. Run Streamlit app
