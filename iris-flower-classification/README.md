# Iris Flower Classification using Decision Tree

## Objective
To predict the species of iris flower using machine learning.

## Dataset
Built-in Iris dataset from Seaborn.

## Features
- Sepal length
- Sepal width
- Petal length
- Petal width

## Algorithm Used
Decision Tree Classifier

## Results
Model achieved good accuracy and tree visualization.

## Tools Used
Python, Seaborn, Scikit-learn, Matplotlib
